Ignore File

  • 有時候，有些檔案不想被加入git追蹤該怎麼辦？
    • 那就忽略它吧！
    • 通常會ignore有機密資料的檔案或暫存檔 
      
echo "filename" > .gitignore
  • .gitignore不用也不能加./在檔案名稱前面
  • 這方法只適用在還沒加入追蹤的檔案

Ignore File

那如果想忽略的檔案已經被加入git追蹤怎麼辦？

先將他從版本庫移除 


git rm --cache "file name"

接著再做一次上一頁的步驟，就可以ignore了

Ignore File

gitignore.io提供了不同語言.gitignore範本
透過git-extras的git ignore-io可以直接從命令列取得範本
另外它的git ignore，也能讓ignore file這件事更輕鬆

Ignore File

有一些file是每個專案都要忽略的，像是vim的暫存檔.swp等檔案
如果每一次都要重新輸入就太麻煩了
這時候就能設定一個global的.gitignore (e.g. ~/.gitignore_global)
然後把這個設定寫進git的config 

git config --global core.excludesfile "path to global gitignore"
